## ğŸ“¦ How to Upgrade

### Using Docker (Most Common)

**Stop your current version:**
```bash
docker stop tome
```

**Pull and start the new version:**
```bash
docker pull ghcr.io/masonfox/tome:0.2.0

docker run -d \
  --name tome \
  -p 3000:3000 \
  -v tome-data:/app/data \
  -v /path/to/calibre/library:/calibre \
  -e CALIBRE_DB_PATH=/calibre/metadata.db \
  ghcr.io/masonfox/tome:0.2.0
```

That's it! Your data is automatically preserved in the `tome-data` volume.

### New Installation

If this is your first time installing Tome:

```bash
docker pull ghcr.io/masonfox/tome:0.2.0

docker run -d \
  --name tome \
  -p 3000:3000 \
  -v tome-data:/app/data \
  -v /path/to/calibre/library:/calibre \
  -e CALIBRE_DB_PATH=/calibre/metadata.db \
  ghcr.io/masonfox/tome:0.2.0
```

Then visit `http://localhost:3000` to get started!
